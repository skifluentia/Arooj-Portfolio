# Arooj Fatima — Portfolio

Personal portfolio website showcasing AI writing, Machine Learning projects, and developer work.

## 🚀 Deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `arooj-portfolio`)
2. Upload all files maintaining this structure:
   ```
   arooj-portfolio/
   ├── index.html
   ├── README.md
   └── work/
       ├── ml-project.html
       ├── ai-banking-article.html
       ├── roundtable.html
       ├── investor-notes.html
       ├── video-portfolio.html
       └── rag-project.html
   ```
3. Go to repository Settings → Pages → Source: Deploy from branch → Branch: main → Save
4. Your site will be live at: `https://yourusername.github.io/arooj-portfolio`

## 📁 File Structure

- `index.html` — Main portfolio homepage
- `work/ml-project.html` — ML projects (Titanic EDA, Diabetes, Iris KNN)
- `work/ai-banking-article.html` — AI in Banking research article
- `work/roundtable.html` — Executive roundtable brief
- `work/investor-notes.html` — Investor strategy framework
- `work/video-portfolio.html` — AI video production
- `work/rag-project.html` — RAG system project
